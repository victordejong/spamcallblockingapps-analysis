Для блокирования сообщений версии Вашей операционной системы Android требуется сделать Call Control приложением обмена сообщениями по умолчанию.

      
  • Многофункциональное приложение обмена сообщениями.
    •   
  • Поддержка SMS
    •   
  • Блокирование мошеннических и фишинговых сообщений.
    •   
  • Блокирование других спам-сообщений.

Не хотите, чтобы Call Control стал приложением для обмена сообщениями? Не беда, Вы все еще можете использовать все функции блокирования звонков, не используя обмен сообщениями.

Хотите попробовать обмениваться сообщениями в Call Control? (Вы всегда сможете переключиться обратно).
