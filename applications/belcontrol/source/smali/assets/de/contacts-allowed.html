Zugelassene Anrufer

Sie können Anrufer auf die Liste der zugelassenen Anrufer hinzufügen, um sicherzustellen, dass sie niemals von Call Control blockiert werden.
Es kann vorkommen, dass unsere Community-Blockierfunktion einen Anruf blockiert, den Sie gerne annehmen möchten. Fügen Sie diese Anrufer hier hinzu, um diese Anrufe zukünftig zu erhalten.
All Ihre Kontakte werden automatisch zugelassen und werden standardmäßig niemals blockiert, so dass Sie diese hier nicht hinzufügen müssen.
