Appels récents — Appels bloqué

Call Control affiche ici tous les appels bloqués.
Call Control affiche également le motif de blocage (par exemple, liste de blocage personnelle ou communautaire) sous le numéro de téléphone.

Astuce: Si vous souhaitez éviter qu’un correspondant soit bloqué, cliquez tout simplement sur Autoriser Correspondant et il pourra toujours vous appeler sur votre téléphone.
