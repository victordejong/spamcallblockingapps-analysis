Ihre Version des Android-Betriebssystems erfordert, dass Sie Call Control als standardmäßige Messaging-App einrichten, um SMS-Nachrichten zu blockieren.

      
  • Voll funktionsfähige Messaging-App.
    •   
  • Unterstützt SMS.
    •   
  • Blockiert Scam- und Phishing-Nachrichten.
    •   
  • Blockiert andere Spam-Nachrichten.

Möchten Sie nicht zu Call Control Messaging wechseln? Keine Sorge, dies ist nur zum Blockieren von Textnachrichten erforderlich. Sie können dennoch alle Funktionen zum Blockieren von Anrufen nutzen.

Möchten Sie Call Control Messaging ausprobieren? (Sie können jederzeit wieder zur alten Anwendung zurückkehren).
